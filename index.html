UKON InfIE

LaTeX Package for Universität Konstanz – Information Engineering and Informatik

generator Download as .zip Download as .tar.gz View on GitHub

General Info

This package is made for students of Information Engineering and Computer Science at the University of Constance.

It contains useful commands for styling the assignments and generating diagrams and graphs (diagrams and graphs not yet imported from the old files).

University and Department are hardcoded into the package but of course changeable.

This package is still in an early alpha stage while our commands and functions are integrated. So as always use at your own risk.

Feel free to use the Issue feature of GitHub to request features or give us feedback or send bug reports.

Installation

The easiest way to get the package is to clone the repository onto your hard drive.

You need to have git installed.

Windows

Clone the repository into your MiKTeX/tex/latex/misc folder:

git clone git://github.com/zinnjonas/ukon-infie.git $PathToMiktex$/tex/latex/misc/ukon-infie

Run a cmd in Admin mode with initexmf --admin --update-fndb

Linux

Clone the repository into your tex-path:

  • global installation: git clone git://github.com/zinnjonas/ukon-infie.git /usr/share/texmf/tex/latex/ukon-infie
  • local installation: git clone git://github.com/zinnjonas/ukon-infie.git ~/texmf/tex/latex/ukon-infie

You now need to update your tex installation. Do this with sudo texhash (global).

Mac OS X

A local installation is recommended for ease of use. If you want to install it global, you should already know how to do that.

First clone the repository into your personal/local tex-path:

git clone git://github.com/zinnjonas/ukon-infie.git ~/Library/texmf/tex/latex/ukon-infie

If your LaTeX installation cannot find the package, run texhash in a console. Then the package should get found properly.

Usage

\documentclass{scrartcl}
\usepackage[lang=de]{ukon-infie}
\Names{Your Name}
\Lecture[FOO]{Foobar}
\Term{WS 1970/1971}     
\begin{document}
    \begin{ukon-infie}[01.01.1970]{42}
        \begin{exercise}[p=10.7]{Title of Exercise}
            \question{Question to be answered?}{
            Answer here.
            }
        \end{exercise}
        \begin{exercise}[p=10]{Title of Exercise}
        Content of Exercise
        \end{exercise}
    \end{ukon-infie}
\end{document}
